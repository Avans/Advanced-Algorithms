Binary Search Trees

In de presentatie hebben we het gehad over wat een Binary Search Tree is, nu gaan jullie er zelf één maken. Eentje die niet alleen maar kan inserten en deleten maar ook nog een hoop extra trucjes kan.

Neem deze broncode over in je eigen project: BST.java BSTNode.java (je mag de code ook porten naar een andere programmeertaal als je dat fijn vindt). Deze code bevat de absolute basis voor een Binary Search Tree waar je getallen in kunt opslaan. De insert() methode hebben we jullie cadeau gegeven. Aan jullie de taak om de rest van de methodes te implementeren:

public boolean exists(int number)
Geeft terug of het getal in de boom bestaat. Implementeer dit op een efficiënte manier, dus ga bijvoorbeeld niet alle nodes in de boom één voor één af.

public int min()
Geeft het kleinste getal wat aan de boom is toegevoegd terug.

public int max()
Geeft het grootste getal wat aan de boom is toegevoegd terug.

public int depth()
Geeft aan hoeveel niveaus diep de boom is op zijn diepste punt. De boom in het plaatje rechts is bijvoorbeeld 4 niveaus diep.

public int count()
Geeft het aantal nodes in de boom terug.

public void print()
Print alle getallen in de boom op volgorde van klein naar groot.

public int printInRange(int min, int max)
Zelfde als vorige, maar nu alleen de waardes die groter of gelijk aan min zijn, en kleiner of gelijk aan max. Implementeer dit weer op een efficiënte manier, ga niet simpelweg langs alle nodes.

public Node delete(int number)
Verwijder een getal uit de boom, zorg ervoor dat je boom een geldige binary search tree blijft. Er is een mooie recursieve versie van de delete(). Die geeft de Node terug die op dezelfde plek in de tree komt te staan als de waarde verwijderd zou zijn.

Let op dat voor alle methodes er zowel een recursieve als een iteratieve oplossing is. We vragen jullie om de recursieve oplossing!


Abstract Syntax Tree

De tree datastructuur wordt ook gebruikt door compilers om een representatie op te bouwen van de structuur van broncode, de zogenaamde abstract syntax tree. Hier rechts zie je een voorbeeld van een tree die is geparsed uit de string "(4+5)*7". Geen zorgen, je hoeft geen parser te schrijven (in dit vak tenminste).

Neem eerst de volgende broncode over: AbstractSyntaxTree.java Expression.java Number.java Operand.java (je mag de code ook porten naar een andere taal als je dat wil).

In de main zie je dat er handmatig nodes worden aangemaakt en aan elkaar worden meegegeven om de boom op te bouwen van het plaatje hier rechts. De opdracht is om deze code uit te breiden zodat je aan elke node (in deze code de klasse Expression) kan vragen om zichzelf te printen met de methode toString() en om zichzelf uit te rekenen met de methode evaluate. Hier moet je net als in de vorige opdracht gebruik maken van recursie

In de main staan twee print statements, als je de code goed het uitgebreid met methodes, instance variabelen en implementaties printen die respectievelijk ((4+5)*7) en 63.
