Abstract Syntax Tree

Trees worden ook gebruikt door compilers om een representatie op te bouwen van de structuur van broncode.

Neem deze broncode over: 
    Number vier = new Number(4);
    Number vijf = new Number(5);
    Operand vier_plus_vijf = new Operand('+', vier, vijf);

    Number zeven = new Number(7);
    Operand dat_keer_zeven = new Operand('*', vier_plus_vijf, zeven);

    System.out.println(dat_keer_zeven.toString()); // ((4+5)*7)
    System.out.println(dat_keer_zeven.evaluate()); // 63
En zorg ervoor dat deze werkt.
